## 👑 Shadow King Two-Phase Boss System

The final boss has **TWO distinct phases** that require different strategies:

### Phase 1: Combat Phase
- **Objective**: Reduce Shadow King's HP from 200+ to 0
- **King's damage**: 10-20 per turn (turn-based combat)
- **Your damage**: 10-15 (normal) or 15-25 (blessed sword)
- **Strategy**: Standard combat - attack repeatedly

### Phase 2: Ritual Phase
- **Triggers**: Automatically when King's HP reaches 0
- **Combat stops working**: Attacking now deals **30 damage to YOU**
- **Only solution**: Execute the 4-step ritual perfectly
- **Requirements**: Oath Token, Blessed Sword, Amulet, True Strike knowledge

💡 **Key Insight**: Combat gets you to Phase 2, but only the ritual achieves victory!

## ⚠️ Common Mistakes (How to Avoid Them)

### ❌ Mistake 1: Wear the Crown

- **Problem**: +10 corruption prevents ritual success
- **Fix**: Take it but leave it in inventory. NEVER use `wear crown`

### ❌ Mistake 2: Attack Shadow King directly during Ritual Phase

- **Problem**: During Ritual Phase (after HP = 0), attacking deals **30 damage to YOU** as punishment
- **Fix**: Follow the 4-step ritual sequence instead of attacking
- **Remember**: Combat works in Phase 1, ritual required in Phase 2

### ❌ Mistake 3: Try ritual without Blessed Sword

- **Problem**: True Strike won't work
- **Fix**: Must bless sword at River WITH Magic Amulet present

### ❌ Mistake 4: Skip ritual steps

- **Problem**: Can't complete ritual
- **Fix**: MUST do: remove crown → activate amulet → wait → true strike

### ❌ Mistake 5: Forget Oath Token

- **Problem**: **TRIGGERS DARK ENDING IMMEDIATELY** even if everything else is perfect!
- **Fix**: Build guard trust to 4/5, then talk for token. This is MANDATORY!

### ❌ Mistake 6: Encounter Kael without Sword

- **Problem**: Can't defeat him
- **Fix**: Must have Sword in inventory (HP ≥ 60 also needed)

### ❌ Mistake 7: Let health drop below 60 before Kael

- **Problem**: Kael will refuse to fight (you don't meet requirements)
- **Fix**: If HP < 60, go to Village and rest (+30 HP, one-time)

### ❌ Mistake 8: Waste Village rest at full HP

- **Problem**: Can't rest when HP = 100, wastes opportunity
- **Fix**: Save rest for after Kael fight or when HP is actually low

### ❌ Mistake 9: Think bridge repair unlocks movement

- **Problem**: Confusion about why you still can't move north
- **Fix**: Repair gives trust + ritual requirement. Must DEFEAT Kael to unlock path!

**How to prevent:**
- Build guard trust to 4+ (do 4 of 5 trust actions)

- **This is the #1 cause of Dark Ending!**

### 2. 👑 Wearing the Golden Crown

**What happens:**
```
The crown's corruption prevents the ritual.
You must remove it.
```

**How to prevent:**
- Take the crown for points if you want
- NEVER use `wear crown` command
- If wearing it, use `remove crown` immediately

### 3. ⚔️ No Blessed Sword

**What happens:**
```
The Shadow King cannot be defeated without the Blessed Sword.
```

**How to prevent:**
- Get Sword from Dark Dungeon
- Get Magic Amulet from Wizard Tower
- Go to River location
- Use `bless sword` command

### 4. 🛡️ Bridge Not Repaired

**What happens:**
```
The path is not clear. Some trials remain.
```

**How to prevent:**
- Go to Ancient Bridge
- Use `repair bridge` command
- This gives +1 trust AND is required for ritual success

### 5. 💥 Didn't Defeat Kael

**What happens:**
```
The path is not clear. Some trials remain.
```

**How to prevent:**
- Have Sword + HP ≥ 60
- Go to Ancient Bridge
- Use `attack kael` command
- Win the turn-based combat

### 6. 📜 No True Strike Knowledge

**What happens:**
```
The True Strike technique is required to seal the victory.
```

**How to prevent:**
- Build guard trust to 4+
- Talk to Captain Ardyn
- Learn `true strike` from him
